   MFC（Microsoft基础类）库：X-监控项目概述
================================================== =============================

在X-Monitor应用程序已经由应用程序向导创建的。此应用程序不仅显示了MFC的基本用法，也是起点，创建应用程序。

此文件包含的每个文件的构成在X-Monitor应用程序内容的摘要。

的X Monitor.vcxproj
    这是由应用程序向导生成的VC ++项目的主项目文件。而有关的Visual C ++生成该文件的版本信息，该平台已在应用程序向导，配置，以及有关项目功能的信息已被列入选择。

的X Monitor.vcxproj.filters
    这是由应用程序向导生成的过滤器文件的VC ++项目。本文件包含有关文件，并在项目中的过滤器之间的关联信息。此关联，它是用来在IDE为了显示的文件的分组具有相同的扩展名中的特定节点（例如，“。.cpp的”文件与“源文件”过滤器相关联）。

的X Monitor.h
    这是应用程序的主头文件。
    其他项目特定的头（如RESOURCE.H）包含，您声明CXMonitorApp应用程序类。

的X Monitor.cpp
    这是主应用程序源文件，包含了CXMonitorApp应用程序类。

的X Monitor.rc
    这是所有程序使用的Microsoft Windows资源的列表。存储在RES子目录图标，我们已包括位图，和光标。这个文件，你可以直接编辑在Microsoft Visual C ++。项目资源位于1041。

水库\的X Monitor.ico
    这是图标文件，这是作为应用程序的图标。此图标包含在主资源文件的X Monitor.rc。

水库\ XMonitor.rc2
    此文件包含不由Microsoft Visual C ++编辑的资源。所有不能在资源编辑器进行编辑的资源，必须被包括在这个文件中。

应用程序向导将创建对话框类之一。

的X MonitorDlg.h，X-MonitorDlg.cpp - 对话
    这些文件包含CXMonitorDlg类。此类定义应用程序的主对话框的行为。对话框模板是包含在X Monitor.rc可以编辑在Microsoft Visual C ++中。

其他标准文件：

StdAfx.h中，StdAfx.cpp
    这些文件用于建立名为X-Monitor.pch预编译头文件（PCH），预编译类型文件名为StdAfx.obj的。

RESOURCE.H
    这是定义新的资源ID的标准头文件。阅读这个文件的更新，你已经做了微软的Visual C ++。
